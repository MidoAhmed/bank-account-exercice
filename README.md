# bank-account-exercice

# Utilisation

1. Cloner le projet

```bash
    git clone https://github.com/MidoAhmed/bank-account-exercice.git
```

2.  Installez les dépendances:

```bash
    cd bank-account-exercice
    npm install
```

3. Exécuter les tests

```bash
    npm test
```

# Info :

cette exercice a été réalisée:

- Typescript
- Jest (pour les tests)
- ESLint + prettier (QA)
